# Pattern-Maker

## Set Up Venv
1. `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process`
1. `.venv\Scripts\Activate`
1. `py main.py`

## Process for Creating Icons/Symbols
1. https://www.flaticon.com
1. Copy Paste into Paint.Net
1. Resize Canvas to 768
1. Resize Image to 512
1. Posterize to one color
1.  Save image
